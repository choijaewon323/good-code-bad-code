# 6.5 오해를 일으키는 함수는 작성하지 말라

- displayLegalDisclaimer() : 호출 시 법적 고지 사항이 표시된다고 가정

## 6.5.1 중요한 입력이 누락되었을 때 아무것도 하지 않으면 놀랄 수 있다

```java

void displayLegalDisclaimer(String? legalText) {
    if (legalText == null) {
        return;
    }

    ...
}

```

- 항상 display할거라고 생각했는데, legalText가 없으면 법적 고지를 하지 않는다 -> 위법이 될 수 있음

## 6.5.2 해결책: 중요한 입력은 필수 항목으로 만들라

- legalText가 반드시 null이 아니도록 설정한다.

```java

void displayLegalDisclaimer(String legalText) {
    if (legalText == null) {
        return;
    }

    ...
}

```
