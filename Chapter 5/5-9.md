# 5.9 프로그래밍 언어의 새로운 기능을 적절하게 사용하라

## 5.9.1 새 기능은 코드를 개선할 수 있다

- 전통적 자바 코드로 작성된 리스트 필터링
```java

List<String> getNonEmptyStrings(List<String> strings) {
    List<String> nonEmptyStrings = new ArrayList<>();

    for (String str : strings) {
        if (!str.isEmpty()) {
            nonEmptyStrings.add(str);
        }
    }

    return nonEmptyStrings;
}
```

- 스트림을 써서 개선
```java
List<String> getNonEmptyStrings(List<String> strings) {
    return strings
    .stream()
    .filter(str -> !str.isEmpty())
    .collect(toList());
}

```

- 훨씬 간략해짐

## 5.9.2 불분명한 기능은 혼동을 일으킬 수 있따

- 스트림이 좋지만, 아직 다른 개발자들이 스트림에 안 익숙다면 안 쓰는게 나을 수도 있다.

## 5.9.3 작업에 가장 적합한 도구를 사용하라

- 새로운 기능이 언제나 최상의 방법은 아닐 수 있다.
- 만약 맵에서 어떤 키에 대한 값을 찾는다면? String value = map.get(key);
  - 이거 한줄로 해결됨
- 근데 스트림으로 굳이 쓴다면?
```java

Optional<String> value = map.entrySet()
.stream()
.filter(entry -> entry.getKey().equals(key))
.map(Entry::getValue)
.findFirst();

```

- 굳이 이렇게 쓸 필요가 있을까 싶지만 하나에 매몰되면 이렇게 짤 수도 있다.
- 항상 내가 할 작업에 적합한 도구를 사용해야한다
